Report on Ayurvedic Remedies and Indian Home Portal

1. Introduction

The "Ayurvedic Remedies and Indian Home Portal" project aims to provide users with easy access to various Ayurvedic remedies for common health issues. The portal features a user-friendly interface that allows users to navigate through different symptoms and find corresponding remedies. The project utilizes HTML, CSS, and JavaScript to create an interactive experience.

2. Literature Survey

Ayurvedic medicine is a traditional Indian system of medicine that emphasizes natural healing and holistic health. Various online platforms provide information on Ayurvedic remedies, but many lack interactivity and user engagement. This project aims to fill that gap by offering a visually appealing and easy-to-navigate portal.

3. Methodology

The project was developed using:

  • HTML: For structuring the content and layout of the portal.
  • CSS: For styling the elements, ensuring a responsive design that adapts to different screen sizes.
  • JavaScript: For implementing interactive features, such as a carousel for navigating through remedies.

The coding practices followed include modular design, where each section of the portal is encapsulated in its own HTML file, and the use of external stylesheets for better maintainability.

4. Result and Discussion

The portal successfully displays various symptoms with corresponding images and links to remedies. The carousel feature enhances user experience by allowing easy navigation through different remedies. The responsive design ensures accessibility on various devices, making it user-friendly.

5. Conclusion and Future Work

The "Ayurvedic Remedies and Indian Home Portal" serves as a valuable resource for individuals seeking natural remedies for common health issues. Future work could include adding more symptoms and remedies, integrating user feedback mechanisms, and enhancing the visual design with animations.
